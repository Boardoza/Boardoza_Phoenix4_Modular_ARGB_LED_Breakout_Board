# Boardoza Phoenix4 – Breakout Board

The **Boardoza Phoenix4** is a modular addressable RGB LED breakout board designed to help makers create custom lighting patterns with maximum flexibility. Built around **addressable RGB LEDs**, this board integrates four addressable RGB LED segments in a compact and daisy-chain form factor.

The board allows multiple units to be soldered together to form letters, numbers, symbols, or custom lighting shapes. It supports both **3.3V and 5V power supplies** and is compatible with a wide range of microcontrollers and digital systems. The **Boardoza Phoenix4** is suitable for decorative lighting, interactive displays, prototypes, and general-purpose addressable LED applications.

## Key Features

- **Modular ARGB Lighting Design:** Allows multiple boards to be soldered together to create letters, numbers, or custom shapes.
- **Addressable RGB LEDs:** Each LED can be individually controlled for precise color and brightness adjustments.
- **Wide Voltage Compatibility:** Supports both **3.3V and 5V** power supplies for broad microcontroller compatibility.
- **TTL & CMOS Logic Support:** Ensures seamless operation with different digital logic levels.
- **Expandable Structure:** Designed for daisy-chaining to build larger and more complex lighting systems.

## Technical Specifications

**Model:** Phoenix4   
**Manufacturer:** Boardoza  
**Functions:** Addressable RGB LED Lighting Board  
**Input Voltage:** 3.3 V / 5 V  
**LED Model:** T3A33BRG-H9C0002X1U1930  
**Number of LEDs:** 24   
**LED Type:** Individually addressable RGB LEDs  
**Logic Compatibility:** TTL & CMOS  
**Data Interface:** Serial Data & Clock (DIN / CIN / DOUT / COUT)  
**Daisy-Chain Support:** Yes  
**Operating Temperature:** -40 °C to +85 °C  
**Board Dimensions:** 6.9 mm × 34 mm
